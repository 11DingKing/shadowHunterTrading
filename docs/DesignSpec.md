# Shadow-Hunter v10.4 设计规范

## 1. 设计理念

### 1.1 主题定位
**地狱深渊风格** - 暗黑、神秘、科技感

### 1.2 设计关键词
- 深邃 (Deep)
- 熔岩 (Lava)
- 幽冥 (Shadow)
- 精密 (Precision)

---

## 2. 色彩系统

### 2.1 主色板

| 色彩名称 | HEX | RGB | 用途 |
|----------|-----|-----|------|
| 深渊黑 | `#1e1e1e` | 30,30,30 | 主背景 |
| 暗夜灰 | `#2d2d2d` | 45,45,45 | 卡片背景 |
| 边界灰 | `#3d3d3d` | 61,61,61 | 边框/分割线 |
| 迷雾灰 | `#666666` | 102,102,102 | 次要文字 |
| 幽灵白 | `#e0e0e0` | 224,224,224 | 主要文字 |

### 2.2 强调色

| 色彩名称 | HEX | RGB | 用途 |
|----------|-----|-----|------|
| 青焰 | `#00bcd4` | 0,188,212 | 主强调/选中态 |
| 熔岩红 | `#ff3d00` | 255,61,0 | 警告/危险/输 |
| 翠金 | `#4caf50` | 76,175,80 | 成功/赢 |
| 琥珀 | `#ffc107` | 255,193,7 | 警示/影子模式 |
| 紫晶 | `#9c27b0` | 156,39,176 | 特殊状态 |

### 2.3 状态色彩映射

| 状态 | 背景色 | 文字色 | 图标 |
|------|--------|--------|------|
| WARMUP (预热) | `#2d2d2d` | `#ffc107` | ⏳ |
| REAL (实盘) | `#1b3a1b` | `#4caf50` | 🎯 |
| SHADOW (影子) | `#3d2d1d` | `#ffc107` | 🛡️ |
| WIN (胜利) | `#1b3a1b` | `#4caf50` | ✅ |
| LOSE (失败) | `#3a1b1b` | `#ff3d00` | ❌ |

## 3. 字体系统

### 3.1 字体族

```
主字体: "Microsoft YaHei", "PingFang SC", "Noto Sans CJK SC", sans-serif
等宽字体: "JetBrains Mono", "Consolas", monospace
```

### 3.2 字号规范

| 级别 | 大小 | 权重 | 用途 |
|------|------|------|------|
| H1 | 24px | Bold | 页面标题 |
| H2 | 18px | Bold | 区块标题 |
| H3 | 16px | Medium | 卡片标题 |
| Body | 14px | Regular | 正文内容 |
| Small | 12px | Regular | 辅助信息 |
| Tiny | 10px | Regular | 时间戳/标签 |

## 7. 动效规范

### 7.1 过渡时间

| 类型 | 时长 | 缓动函数 |
|------|------|----------|
| 快速 | 150ms | ease-out |
| 标准 | 250ms | ease-in-out |
| 慢速 | 400ms | ease-in-out |

### 7.2 动效场景

| 场景 | 动效 |
|------|------|
| 按钮hover | 背景色渐变 150ms |
| 页面切换 | 淡入淡出 250ms |
| 状态变更 | 背景闪烁 + 渐变 400ms |
| 胜负结果 | 按钮放大1.1x + 回弹 |
